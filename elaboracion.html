MOTORES

  • Iniciamos con los motores, estos le añadimos las aspas para que se ejecute la respectiva generacion de energia. Realizamos una conexiòn en paralela entre los motores para aumentar la potencia de generaciòn, posteriormente los aderimos a una base para mantener estabilidad y producciòn.

MAQUETA DE REFERENCIA

  • Esta maqueta está diseñada para demostrar el funcionamiento de la energía eólica en una ciudad. Cuenta con hélices conectadas a motores que generan electricidad a partir del viento, la cual se distribuye a través de cables para alimentar postes de luz y una cancha de fútbol iluminada. Además, representa un entorno urbano con casas, calles y alumbrado público, mostrando cómo la energía renovable puede integrarse en la vida cotidiana para suministrar electricidad de manera sostenible.

PROTOBOARD

  • El protoboard contara con las conexiones como; el Arduino nano que realizara las instruciones para el funcionamiento del proyecto, jumpers que enviaran la corriente a la bateria que la almacenara, mòdulo bluetooth para conectar el arduino al mòvil.

PROGRAMA

  • ARDUINO: Está programado para realizar las instrucciones como el encendido de focos led y la activación de sonido de la alarma a través de la conexiòn bluetooth con el móvil, el arduino enviara corriente a traves de sus pines a los diferentes componentes.
  • APP INVENTOR: Es una app programada que funciona en conjunto con el arduino, está mandara señales al arduino para que este realice las instrucciones.

PRODUCTO FINAL

  • Una vez concluido con el proyecto, obtenemos una casa intelignet que funcionara a través de energía eólica, esta contara con un sistema de activación de voz por medio de la app del mòvil y podra realizar las tareas automáticamente.

Ilustrativa y resumen del funcionamiento del circuito

  • Este proyecto consiste en el diseño y desarrollo de una casa domótica a escala, controlada a través de una aplicación móvil desarrollada en MIT App Inventor, que permite al usuario activar funciones tanto manualmente como mediante comandos de voz. El sistema está basado en Arduino UNO, sensores, servomotores y componentes electrónicos básicos, con la particularidad de estar alimentado por energía eólica generada por turbinas caseras. Funciones principales: Control de Iluminación
      Cada LED representa la luz de un ambiente:
  • Entrada
  • Cocina
  • Sala
  • Dormitorio
  • Garaje
  • Estas luces pueden encenderse/apagarse desde la app o por comando de voz.
      • Sensor de lluvia
  • Detecta la presencia de agua de lluvia.
  • Al activarse:
  • Mueve un servo que simula guardar un tendedero.
  • Enciende un LED de alerta que parpadea 3 veces.
  • Enciende un buzzer que suena 3 veces.
      • Ventana automática
  • Dos servos simulan la apertura/cierre de una ventana.
  • Se controlan por comandos desde la app o voz.
      • Simulación de cableado eléctrico
  • Representado con cables de timbre y leds.
  • Alimentado por 4 turbinas de viento que simbolizan una red eléctrica autosostenible.
  • Proporciona energía a todo el sistema domótico.
      • Energía renovable
  • Todo el sistema es alimentado por motores con turbinas que generan energía eólica.
  • Promueve el uso de fuentes limpias y sostenibles.

Conclusiones:

  • Este proyecto de casa domótica combina control mediante app y comandos de voz con energía eólica para crear un hogar autosostenible. Utiliza sensores, servomotores y Arduino para gestionar funciones como iluminación y ventanas. Nos emocionó ver cómo nuestra idea cobraba vida, destacando la importancia de la energía renovable y cómo puede transformar nuestros hogares hacia un futuro más sostenible.
